# Bored

This webapp consists of two mini applications. The avatar generator on the home page
uses [DiceBear Avatars](https://avatars.dicebear.com/) API and npm module. It allows you to create a cutomized pixel avatar or generate a random one. 
The Bored page uses [Bored API](https://www.boredapi.com/) that generates a random activity for you or your friends to do.
The tech stack includes express, ejs, css, bootstrap. [NES.css](https://nostalgic-css.github.io/NES.css/) is used for style.
